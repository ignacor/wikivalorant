Vocabulario clave para Valorant

En esta seccion te mostraremos cuales son las palabras claves que necesitas saber antes de jugar valorant, para tener una optima comunicacion con tu equipo,la comunicacion es una de las cosas mas importantes en valorant.

Eco

Se refiere a ahorrar,es decir hacer una media compra o ni siquiera comprar,normalmente se utiliza cuando hay muy poca economia en todo el equipo para poder tener mejores armas en la siguiente ronda.

Retake

El Retake se utiliza en la defensa,es una estrategia para recuperar el site,basicamente es retroceder y esperar a tu equipo para poder defender todos juntos.

Rush

Es otro tipo de estrategia que se utiliza en ataque,cuando alguien quiere "rushear" significa que quiere entrar al site muy rapido utilizando todas las habilidades.

Defusando.

Se refiere a que el enemigo esta desactivando la spike.

Pekea

Accion de asomar y buscar el duelo con el enemigo.

Corta

Es una zona del mapa,como su nombre indica es la mas "corta" o chica que hay.

WallBang

Es una estructura del mapa en la que las balas traspasan la misma.
